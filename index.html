About Wryt

Wryt is a powerful, free alternative to Grammarly that leverages multiple AI providers to deliver exceptional grammar and spell checking. Our intelligent waterfall failover system ensures you always get the best results by automatically switching between providers like Google Gemini, Groq, OpenRouter, Cerebras, and SambaNova.

Unlike traditional grammar checkers, Wryt specializes in correcting speech-to-text errors, making it perfect for users who dictate their content. It understands context, catches homophones, and transforms your dictated thoughts into polished, professional writing.

Installation

Manual Installation (Chrome/Edge/Brave):

  1. Clone or download this repository from GitHub
  2. Open Chrome and navigate to chrome://extensions/
  3. Enable Developer Mode (toggle in top-right corner)
  4. Click Load unpacked and select the extension folder
  5. The Wryt icon should appear in your browser toolbar
  6. Click the extension icon and configure your AI provider API keys in the settings

How to Use

  1. Configure API Keys: Click the Wryt extension icon and add your API keys for at least one AI provider (Gemini, Groq, OpenRouter, etc.)
  2. Select Text: Highlight any text on a webpage
  3. Right-Click: Choose "Check Grammar & Spelling" from the context menu
  4. Review Suggestions: View color-coded corrections (Red: Critical Grammar, Blue: Clarity, Green: Engagement, Purple: Tone)
  5. Generate Text: Right-click in any text field and select "Write with AI..." to generate content
